# Nome do Projeto
LOJA DO HARDWARE (Alfa)

## Tecnologias Utilizadas

- React
- Express
- MySQL
- Node.js

## Como Iniciar a Aplicação

1. Clone o repositório para a sua máquina local usando `git clone URL_DO_REPOSITORIO`.
2. Navegue até a pasta do projeto usando `cd NOME_DA_PASTA`.
3. Instale todas as dependências usando `npm install`.
4. Inicie a aplicação usando `npm start`.

## Configurando o .env

Crie um arquivo `.env` na raiz do projeto e adicione as seguintes variáveis de ambiente:

```env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=admin
DB_NAME=loja_do_hardware
